# IPL Data Analysis

## Project Overview

This project analyzes Indian Premier League (IPL) cricket data using Python, Pandas and Matplotlib.
## Project Screenshots

The project provides insights into:

- Team Performance
- Top Run Scorers
- Top Wicket Takers
- Toss Impact
- Venue Analysis
- Season-wise Statistics

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib

---

## Dataset

- matches.csv
- deliveries.csv

Dataset Source: Kaggle IPL Dataset

---

## Key Features

. Team Win Analysis

. Toss Impact Analysis

. Top Batsmen Analysis

. Top Bowlers Analysis

. Venue Performance Analysis

. Season-wise Insights
